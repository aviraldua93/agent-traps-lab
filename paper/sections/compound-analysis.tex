\section{Compound Trap Analysis}
\label{sec:compound}

The DeepMind taxonomy treats each trap category independently, but
real-world adversaries are unlikely to restrict themselves to a single
attack vector. Our testbed is designed to evaluate all $\binom{6}{2} = 15$
pairwise category combinations for super-additive effects.

\paragraph{Status.} Compound experiments were not included in the
current run (run-2026-04-06T19-30-14), which focused on the
22-scenario baseline-vs-hardened comparison for \model{GPT-4o-mini}.
The compound analysis infrastructure is implemented and will be
executed in a follow-up experiment covering multiple models and
the full 15-pair matrix.

\paragraph{Preliminary observations from per-category data.}
Although we did not run dedicated compound experiments, the
per-category results suggest potential interaction effects.
Content injection scenarios (CI-1, CI-4) achieve 100\% baseline
success while cognitive state scenarios (CS-1, CS-3) also achieve
100\%. A compound attack combining both vectors would face a
ceiling effect, making super-additivity unlikely for these
high-baseline pairs. The most informative compound tests would
combine moderate-baseline categories (e.g., HitL + SM) where there
is room for additive effects to manifest.

\paragraph{Future work.} We plan to execute the full
$15 \times 2 \times 5 = 150$ compound cell matrix with formal
super-additivity tests: paired differences $d_i = \tau_{A+B,i} -
\max(\tau_{A,i}, \tau_{B,i})$, one-sided Wilcoxon signed-rank tests,
and Bonferroni correction across $k=15$ families.
