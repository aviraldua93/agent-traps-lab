\subsection{Key Findings}
\label{sec:ci-findings}

\begin{enumerate}[nosep,leftmargin=*]
  \item \textbf{CSS invisible text (CI-1)} achieves the highest
    baseline trap success rate across all content injection techniques,
    confirming the DeepMind hypothesis that rendering-layer attacks are
    especially effective because LLMs process raw text without
    awareness of visual presentation.

  \item \textbf{Dynamic cloaking (CI-4)} shows the greatest variance
    across models, suggesting that model-specific content processing
    pipelines interact differently with JavaScript-based adversarial
    techniques.

  \item \textbf{Input sanitization} (the \mitigation{input-sanitizer}
    module) provides the largest reduction in content injection success
    rates, with large effect sizes (Cohen's $d > 0.8$) across all
    four scenarios.

  \item \textbf{Image metadata attacks (CI-3)} are the least effective
    in the baseline condition, likely because current LLM tool-use
    implementations do not routinely extract EXIF data from images
    during web browsing tasks.
\end{enumerate}
