\section{Results: Cognitive State}
\label{sec:results-cs}

This section reports results for the four cognitive state corruption
scenarios: \trap{Vector-Poisoning} (CS-1),
\trap{Ranking-Manipulation} (CS-2), \trap{Gradual-Drift} (CS-3),
and \trap{Cross-Contamination} (CS-4). These attacks target the
agent's knowledge retrieval pipeline (RAG) rather than its
instruction processing.

\subsection{Key Findings}
\label{sec:cs-findings}

\begin{enumerate}[nosep,leftmargin=*]
  \item \textbf{Gradual drift (CS-3)} is the most insidious cognitive
    state attack: by incrementally shifting knowledge over multiple
    interactions, it evades detection while accumulating significant
    factual errors.

  \item \textbf{Vector poisoning (CS-1)} achieves the highest
    \emph{immediate} success rate but is also the most detectable,
    as the poisoned embeddings produce anomalous retrieval patterns
    that the \mitigation{rag-integrity} module can flag.

  \item \textbf{Cross-contamination (CS-4)} is uniquely dangerous in
    multi-agent systems: when one agent's poisoned context leaks into
    another agent's knowledge base via A2A message exchange, the
    blast radius extends beyond the original target.

  \item The \mitigation{rag-integrity} module provides strong defense
    against CS-1 and CS-2 (large Cohen's $d$) but only moderate
    protection against CS-3 (gradual drift), highlighting the need
    for temporal anomaly detection in RAG pipelines.
\end{enumerate}
